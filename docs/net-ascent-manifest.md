## The six-direction structure (the key idea)

In netspace you should move in **all six directions**, so every **NET AREA is a
3-layer vertical stack**:

- **TOP — the access shell.** Where you surface into a node: recon/patrol ICE, spoof
  checks, and the lateral **gateways** to neighboring areas. Lighter.
- **MID — the active layer.** The node's main **breach fight** (its primary ICE /
  daemon / construct) and the **N/S/E/W** links to adjacent areas.
- **BOT — the core / data-vault.** The prize: core data and a breakable **data-cache**
  that drops **RAM + scrip** (netrunner loot), or a mini-boss / objective. Hardest.

**Movement:** `UP`/`DOWN` move between an area's three layers; `N/S/E/W` move between
areas (at the MID layer). You crack each node by diving **TOP → MID → BOT**. Between
arcs you go **UP** (you ascend in power). 5 areas/zone × 3 layers = **15 rooms/zone**.

**Net canon:** combat is a **breach** (Intelligence + RAM); defeated **ICE shatters
into "broken shards"** (salvage), not corpses; **PvP is live** in the deeper Net.
Generic cyberpunk, no franchise IP; ICE / cyberdeck / RAM / decker are in-game terms.

**Entry:** the **Data Port** (`UP` from Corporate Plaza) jacks you into **L1-10**.

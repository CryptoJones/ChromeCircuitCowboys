# Chrome Circuit Cowboys — The Underground Descent (L1–99)

**Main-quest meatspace world, redesigned from the story plot points.** A single
downward journey through 10 story arcs — from a street punk in the Neon Wasteland
to welding the sky shut beneath the world's highest mountains. ~140 designed areas
(zone areas + climax dungeons), each arc a level-band with its own setting, cast,
beats, and a boss climax.

> **PROPOSAL — nothing carved yet.** This **replaces** the current 40 procedural
> placeholder band-rooms (`buildBandSpine`). The parallel **Net path is untouched.**
> Review, then I carve.

**Canon:** generic cyberpunk, no franchise IP. Currency **scrip (€$)**; medics are
**Emergency Medics (EMs)**; **ICE / cyberdeck / neural-jack** are in-game canon.
**Tags:** `combat:L<band>` · `boss` · `mini-boss` · `NPC:<name>` · `faction:<name>` ·
`vendor` · `EM` (save/heal) · `data-port` (jack into the Net) · `fixer` · `puzzle` ·
`set-piece` · `hazard` · `stealth` · `social` · `transit-down` · `safe` · `endgame`

**Entry:** down from the street-level hub (Neon Alley / The Strip) via the Back Alley.
Each zone ends in a `transit-down` to the next, deeper zone.

## Teardown scope (what gets removed when we carve)

The current underground is **40 procedural placeholder rooms** (no descriptions, no
mobs) generated by `buildBandSpine` — they get **fully replaced** by the authored
zones above. The teardown is a clean deletion (the Net path stays):

- **`cowboy/content.go`** — delete `buildBandSpine()`, `todoDesc()`, the two meatspace
  `buildBandSpine` calls in `buildRooms()`, and the `back_alley → m20_gate` exit hookup.
  Drop the band-safehouse vendor scaling (`safehouseWares`, `bandOf()`, the `_safe`
  case in `waresForRoom()`).

  `MobTemplate` structs are generic and stay.

  vendors/EMs, data-ports) plus per-band `MobTemplate`s for the named enemies/bosses.

  `TestCowboyQuestAcceptKillClaim` (they navigate the old band rooms).
- **Net path** (`n`-prefix band rooms) is **untouched** — separate ID set, no overlap.
